## Failure-Mode Reflection

The failure mode I noticed most was short memory. At first, the project felt confusing because the assignment was not just asking for a normal website. It was asking for a scrollytelling page while also practicing a process for working with AI. To fix that, I wrote the project direction into files instead of relying only on the chat. The README explains the final project, and this PROCESS.md explains how I worked. This helped make the project more organized and easier to continue.
