# Wages vs. Inflation Scrollytelling Project

- Topic: Wages vs. Inflation

## Project Description

This project is a small scrollytelling website based on my wages vs. inflation data story. The goal is to explain how wages can increase over time while still not fully improving workers' purchasing power because of inflation.

## Main Question

Did wages actually improve, or did inflation reduce their real value?

## Claim

Although wages increased over time, inflation reduced the real improvement workers experienced.

## Pages

- Homepage: scrollytelling story about wages, inflation, and real wages
- Methodology page: explanation of the project question, claim, process, and exit checks
- Images page: automatically lists images added to the project

## Scrollytelling Technique

The homepage uses a sticky visual panel while the viewer scrolls through different story sections. Each section reveals one part of the argument, moving from the basic question to the final takeaway.

## Spec-Driven Development Process

For this project, I used a spec-driven development process. Instead of asking AI to build the whole project at once, I broke the work into smaller steps:

1. Define the project topic
2. Write the main question
3. Write the claim
4. Plan the scrolling story sections
5. Build the homepage
6. Add a regular methodology page
7. Add an image to the public image library
8. Test the project
9. Deploy with GitHub Pages
